# HNUTest

## API

| URI                | METHOD | DATA              | FUNCTION                                    |
| ------------------ | ------ | ----------------- | ------------------------------------------- |
| `/api/get/score`   | `GET`  | `{stuNo: String}` | 返回所有 `stuNo` 字段与指定字符串相同的记录 |
| `/api/get/student` | `GET`  | `{stuNo: String}` | 返回 `stuNo` 字段与指定字符串相同的记录     |
